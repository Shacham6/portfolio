**NOTE:** The game is currently *work in progress* as I'm working on it in my free time.

A rogue-lite where everything is hackable — doors, enemies, yourself. Features an in-game terminal where every entity has its own filesystem and runnable commands. Built from scratch: immediate-mode GUI, field-of-view, pathfinding, and deep mechanic interactions.

The development of the game is done with **Go** and **Raylib**, and nothing else. Yes, that means:

- No game engine.
- No additional library of any kind.

Everything developed here was handmade across evenings that spanned months.

Here are the features so far:

- **PROCEDURAL GENERATION:** The game maps will have different layouts and different item placements each time.

That's the premise of the game. Technologically, though, there's a lot more that's been happening. The project included:

- Implementing a real-time immediate-mode GUI framework.
- Implementing "vision" for entities via the shadowcast algorithm.
- Implementing `A*` - which is a pathfinding algoritm, to allow entities to navigate through the rooms correctly.

  Everything *from scratch*.
